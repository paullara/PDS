📄 Personal Data Sheet System

The Personal Data Sheet (PDS) System is a web-based application designed to manage and organize employee information in a centralized and digital format. It replaces manual, paper-based records with a structured system that allows efficient storage, retrieval, and updating of employee data.

This system enables users to create, view, update, and delete employee profiles, including personal details, contact information, educational background, and employment history. It is built to improve data accuracy, reduce redundancy, and streamline administrative processes.

🚀 Features
Employee profile management (CRUD)
Organized personal and professional data storage
Search and filtering functionality
User-friendly interface
Secure and centralized data handling
🛠️ Tech Stack
Backend: Laravel
Frontend: React (Inertia.js)
Database: MySQL
🎯 Purpose

The goal of this project is to provide an efficient and reliable system for managing employee records, helping organizations maintain accurate data and improve overall workflow.
